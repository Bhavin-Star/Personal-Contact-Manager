Personal Contact Manager
A simple Python-based Personal Contact Manager that allows users to store, search, view, and delete contact details using an SQLite database.
This project demonstrates basic CRUD (Create, Read, Update, Delete) operations using Python and SQLite and is intended for learning and academic purposes.

Features
Add new contacts
View all saved contacts
Search contacts by name
Search contacts by phone number
Delete contacts
Display contacts in ascending order
Persistent storage using SQLite database

Technologies Used
Python 3
SQLite3 (Python built-in library)
Project Structure
Personal-Contact-Manager/

app.py
Main application file

additems.py
Adds new contact to the database

delete.py
Deletes a contact from the database

searchByName.py
Searches contact by name

searchByNo.py
Searches contact by phone number

printDatainAscending.py
Displays contacts in ascending order

viewdb.py
Displays all stored contacts

data.db
SQLite database file

How to Run the Project
Clone the repository
git clone https://github.com/Bhavin-Star/Personal-Contact-Manager.git
Move into the project directory
cd Personal-Contact-Manager
Run the main application
python app.py
Follow the terminal instructions to manage contacts

Database
The project uses an SQLite database named data.db
The database is automatically updated when scripts are executed
